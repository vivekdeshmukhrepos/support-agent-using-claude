Complete reference for all available API endpoints in the Email Support Agent

🌐 Base URL

http://localhost:8000

1️⃣ Health Check

GET /health

Check if the API server is running and healthy.

📤 Response

{
  "status": "ok"
}

🧪 Try It

2️⃣ Submit Email for Processing

POST /api/emails

Submit a customer email for processing through the AI agent pipeline.

📥 Request Body

{
  "sender": "customer@example.com",
  "recipient": "support@company.com",
  "subject": "Unable to reset my password",
  "body": "I tried to reset but didn't receive the email"
}

📤 Response

{
  "email_id": "email_abc123",
  "status": "sent",
  "message": "Email processed successfully. Status: sent"
}

📝 Parameters

Parameter Type Required Description
sender string Yes Sender email address
recipient string Yes Recipient email address
subject string Yes Email subject
body string Yes Email message body

🧪 Try It

3️⃣ Get Email Status

GET /api/emails/{email_id}

Retrieve the status and details of a processed email.

📥 Path Parameter

Parameter Type Description
email_id string Email identifier (from POST response)

📤 Response

{
  "email_id": "email_abc123",
  "status": "sent",
  "intent": "question",
  "urgency": "high",
  "category": "account",
  "confidence": 0.92,
  "final_status": "sent",
  "escalated": false,
  "escalation_ticket_id": null,
  "draft_response": "Hello! I understand you're having trouble..."
}

🧪 Try It

💻 cURL Examples

Health Check

curl http://localhost:8000/health

Submit Email

curl -X POST http://localhost:8000/api/emails \
  -H "Content-Type: application/json" \
  -d '{
    "sender": "customer@example.com",
    "recipient": "support@company.com",
    "subject": "I need help",
    "body": "Can you help me reset my password?"
  }'

Get Email Status

curl http://localhost:8000/api/emails/email_abc123

List All Emails

curl http://localhost:8000/api/emails-list

📋 HTTP Status Codes

Code Meaning Example
200 Success Email retrieved successfully
404 Not Found Email ID doesn't exist
422 Validation Error Invalid email format
500 Server Error Processing failed
